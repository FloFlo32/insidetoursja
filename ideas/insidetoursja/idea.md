# Idea: Inside Tours Jamaica

- **Reference URL:** https://insidetoursja.com/
  _(brand color, logo, fonts, name, ALL content, copy, images, and pages are scraped
  from here. Content, structure, images, and logo must be preserved exactly.)_
- **Design inspiration URL:** https://adventurecruises.getyetti.com/
  _(layout, spacing, typography, and premium modern feel only — do NOT pull content,
  copy, or images from this site, reference for design/layout only)_
- **What we're building:** A full redesign of insidetoursja.com (a Jamaica tours /
  excursions company) with the exact same content, pages, copy, images, tour listings
  and logo, but rebuilt with an ultra-modern, clean, card-based UI inspired by the
  layout/spacing/typography of adventurecruises.getyetti.com.
- **Audience:** Travelers booking Jamaica tours, excursions, and airport transfers.
- **Tone / vibe:** Premium, clean, light, sophisticated, trustworthy travel/tourism brand.
- **Register:** design SERVES the existing content — do not rewrite or omit any copy.
- **Must-have sections:** hero (every page), tour/excursion cards, testimonials,
  about, contact, FAQ.
- **Must-have pages:** Exactly the same pages/menu structure as insidetoursja.com
  (home + every inner page/tour page it has). Do not add, remove, or rename pages.
- **FAQ widget topics:** Jamaica tours, excursions, airport transfers, bookings, pricing.
- **Anything to avoid:** Dark mode/dark theme, stock images without people, inverted/white
  logo, using the logo as a decorative image inside page content (logo only in navbar +
  footer), placeholder colored boxes, generic AI-template look.

## Design requirements (strict)
- Light theme only. Extract primary/secondary accent colors from the insidetoursja.com
  logo colors.
- Ultra-modern, spacious, card-based UI: rounded corners (12-16px), soft shadows, hover
  lift micro-interactions.
- Breathtaking hero section on every single page using the site's real imagery.
- Use only images that feature people (tours/excursions imagery), matching the same
  images on the same pages/sections as the source site.
- Tour images must match https://insidetoursja.com/ tours exactly.
- Logo used in navbar + footer only, exactly as-is (never inverted to white). Logo also
  used as the favicon.
- Modern sticky nav + organized multi-column footer.

## Images

Logo and content images are scraped directly from https://insidetoursja.com/ via
`npm run ingest`. Design/layout reference only (no asset copying) from
https://adventurecruises.getyetti.com/.
